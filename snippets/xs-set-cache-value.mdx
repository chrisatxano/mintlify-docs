# <img src="../assets/docuBadge (11).png" alt="" data-size="line"> Set Cache Value <a href="#xs-set-cache-value" id="xs-set-cache-value"></a>

```javascript
redis.set {
  key = ""
  data = ""
  ttl = 0
}
```

| Parameter | Purpose                 | Example                       |
| --------- | ----------------------- | ----------------------------- |
| key       | Cache key identifier    | `"user:123"`, `"session:abc"` |
| data      | Value to store          | `"data"`, `{user: "john"}`    |
| ttl       | Time-to-live in seconds | `300`, `3600`                 |

<details>

<summary>Example</summary>

```javascript
redis.set {
  key = "user:"|add:$user.id
  data = $user.profile
  ttl = 3600
}
```

* Stores a value in cache
* Optional TTL for expiration
* Overwrites existing values

</details>
