## <img src="../assets/docuBadge (11).png" alt="" data-size="line"> to\_minutes <a href="#xs-to_epoch_minute" id="xs-to_epoch_minute"></a>

`value|to_minutes:timezone`

<details>

<summary>Examples</summary>

```javascript
"2024-01-01"|to_minutes:"UTC"    // Returns minutes since Unix epoch
"now"|to_minutes:"UTC"           // Returns current minutes since Unix epoch
```

</details>
