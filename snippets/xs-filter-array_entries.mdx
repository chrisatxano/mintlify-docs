## <img src="../assets/docuBadge (11).png" alt="" data-size="line"> entries <a href="#xsarray_entries" id="xsarray_entries"></a>

`value|entries`

<details>

<summary>Examples</summary>

```javascript
[1,2,3]|entries                // Returns [[0,1],[1,2],[2,3]]
{"a":1,"b":2}|entries         // Returns [["a",1],["b",2]]
```

</details>
