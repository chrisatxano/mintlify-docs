## <img src="../assets/docuBadge (11).png" alt="" data-size="line"> index\_by <a href="#xs-index_by" id="xs-index_by"></a>

`value|index_by:key`

<details>

<summary>Examples</summary>

```javascript
[{"id":1,"name":"a"},{"id":2,"name":"b"}]|index_by:"id"    // Returns {1:{"id":1,"name":"a"},2:{"id":2,"name":"b"}}
```

</details>
