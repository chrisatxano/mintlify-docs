## <img src="../assets/docuBadge (11).png" alt="" data-size="line"> remove <a href="#xs-array_remove" id="xs-array_remove"></a>

`value|remove:item:key:strict`

<details>

<summary>Examples</summary>

```javascript
[1,2,3]|remove:2    // Returns [1,3]
{"a":1,"b":2}|remove:null:"b":true    // Returns {"a":1}
```

</details>
