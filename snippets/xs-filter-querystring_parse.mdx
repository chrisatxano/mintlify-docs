## <img src="../assets/docuBadge (11).png" alt="" data-size="line"> querystring\_parse <a href="#xsquerystring_parse" id="xsquerystring_parse"></a>

`value|querystring_parse`

<details>

<summary>Examples</summary>

```javascript
"name=john&age=25"|querystring_parse    // Returns {name: "john", age: "25"}
"x=1&x=2"|querystring_parse            // Returns {x: ["1", "2"]}
```

</details>
