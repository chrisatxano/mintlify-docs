## <img src="../assets/docuBadge (11).png" alt="" data-size="line"> prepend <a href="#xs-prepend" id="xs-prepend"></a>

`value|prepend:item:key`

<details>

<summary>Examples</summary>

```javascript
[1,2]|prepend:0                // Returns [0,1,2]
{"b":2}|prepend:1:"a"         // Returns {"a":1,"b":2}
```

</details>
