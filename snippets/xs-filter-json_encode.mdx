## <img src="../assets/docuBadge (11).png" alt="" data-size="line"> json\_encode <a href="#xs-json_encode" id="xs-json_encode"></a>

`value|json_encode`

<details>

<summary>Examples</summary>

```javascript
"{\"name\":\"John\"}"|json_encode    // Returns {"name":"John"}
"[1,2,3]"|json_encode                // Returns [1,2,3]
```

</details>
