# <img src="../assets/docuBadge (11).png" alt="" data-size="line"> Remove From End of List <a href="#xs-remove-from-end-of-list" id="xs-remove-from-end-of-list"></a>

```javascript
redis.pop {
  key = ""
} as x7
```

| Parameter | Purpose                | Example                          |
| --------- | ---------------------- | -------------------------------- |
| key       | List key               | `"queue:tasks"`, `"stack:items"` |
| as        | Alias for popped value | `x7`, `last_item`                |

<details>

<summary>Example</summary>

```javascript
redis.pop {
  key = "task_queue"
} as next_task
```

* Removes and returns last element
* Returns null if list is empty
* Reduces list length by 1
* Common for stack operations

</details>
