## <img src="../assets/docuBadge (11).png" alt="" data-size="line"> unique <a href="#xs-unique" id="xs-unique"></a>

`value|unique:key`

<details>

<summary>Examples</summary>

```javascript
[1,2,2,3]|unique    // Returns [1,2,3]
[{"id":1},{"id":1}]|unique:"id"    // Returns [{"id":1}]
```

</details>
