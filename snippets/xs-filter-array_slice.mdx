## <img src="../assets/docuBadge (11).png" alt="" data-size="line"> slice <a href="#xs-array_slice" id="xs-array_slice"></a>

`value|slice:offset:length`

<details>

<summary>Examples</summary>

```javascript
[1,2,3,4]|slice:1:2    // Returns [2,3]
["a","b","c"]|slice:0:2    // Returns ["a","b"]
```

</details>
