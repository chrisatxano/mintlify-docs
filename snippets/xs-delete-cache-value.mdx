# <img src="../assets/docuBadge (11).png" alt="" data-size="line"> Delete Cache Value <a href="#xs-delete-cache-value" id="xs-delete-cache-value"></a>

```javascript
redis.del {
  key = ""
}
```

| Parameter | Purpose             | Example      |
| --------- | ------------------- | ------------ |
| key       | Cache key to delete | `"user:123"` |

<details>

<summary>Example</summary>

```javascript
redis.del {
  key = "temp:"|add:$id
}
```

* Removes key and value from cache
* No effect if key doesn't exist

</details>
