# <img src="../assets/docuBadge (11).png" alt="" data-size="line"> Get Cache Value <a href="#xs-get-cache-value" id="xs-get-cache-value"></a>

```javascript
redis.get {
  key = ""
} as x1
```

| Parameter | Purpose                   | Example             |
| --------- | ------------------------- | ------------------- |
| key       | Cache key to retrieve     | `"user:123"`        |
| as        | Alias for retrieved value | `x1`, `cached_data` |

<details>

<summary>Example</summary>

```javascript
redis.get {
  key = "user:"|add:$user.id
} as user_data
```

* Retrieves stored value
* Returns null if key doesn't exist

</details>
