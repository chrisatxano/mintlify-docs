# <img src="../assets/docuBadge (11).png" alt="" data-size="line"> object <a href="#xs-object" id="xs-object"></a>

`object field_name {schema={}}`

<details>

<summary>Examples</summary>

```javascript
object settings {     // An object field for user settings
  schema = {
    theme: "string",
    notifications: "boolean"
  }
}
object address {      // An object field for address details
  schema = {
    street: "string",
    city: "string",
    zip: "string"
  }
}
```

</details>
