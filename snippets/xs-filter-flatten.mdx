## <img src="../assets/docuBadge (11).png" alt="" data-size="line"> flatten <a href="#xs-flatten" id="xs-flatten"></a>

`value|flatten`

<details>

<summary>Examples</summary>

```javascript
[[1,2],[3,4]]|flatten    // Returns [1,2,3,4]
[1,[2,[3]]]|flatten      // Returns [1,2,3]
```

</details>
