## <img src="../assets/docuBadge (11).png" alt="" data-size="line"> intersect <a href="#xs-array_intersect" id="xs-array_intersect"></a>

`value|intersect:array`

<details>

<summary>Examples</summary>

```javascript
[1,2,3]|intersect:[2,3,4]    // Returns [2,3]
["a","b"]|intersect:["b","c"]    // Returns ["b"]
```

</details>
