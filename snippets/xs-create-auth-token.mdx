## <img src="../assets/docuBadge (10).png" alt="" data-size="line"> Create Authentication Token <a href="#xs-securitycreateuuid" id="xs-securitycreateuuid"></a>

```javascript
    security.create_auth_token {
      dbtable = "97"
      extras = '{"role":"admin"}'|json_decode
      expiration = 86400
      id = 1
    } as $authToken
```

<table><thead><tr><th width="175">Parameter</th><th>Purpose</th><th>Example</th></tr></thead><tbody><tr><td>dbtable</td><td>The ID of the database table that has authentication enabled</td><td>"97"</td></tr><tr><td>extras</td><td>A JSON object that contains any extra information you want to include in the token</td><td><pre><code>    '{"role":"admin"}'|json_decode
</code></pre></td></tr><tr><td>expiration</td><td>The time in seconds that the token should remain valid</td><td>86400</td></tr><tr><td>id</td><td>The ID of the record to use to generate the authentication token</td><td>1</td></tr><tr><td>as</td><td>The variable to store the generated token in</td><td>$authToken</td></tr></tbody></table>
