## <img src="../assets/docuBadge (10).png" alt="" data-size="line"> **For Loop**  <a href="#xs-for" id="xs-for"></a>

```javascript
    for (`$loops`) {
      each as $index {
         // Functions go here
      }
    }
```

<table><thead><tr><th>Parameter</th><th width="319">Purpose</th><th>Example</th></tr></thead><tbody><tr><td>$loops</td><td>The number of iterations in the loop</td><td>10</td></tr><tr><td>$index</td><td>The name of the index variable</td><td>index</td></tr></tbody></table>

Place the functions that run as a part of your loop inside {} brackets after defining the index variable.

<details>

<summary>Example</summary>

![](<../assets/CleanShot 2025-02-27 at 09.51.16@2x.png>)

```javascript
  for (`10`) {
      each as index {
        util.sleep {
          value = 1
        }
      }
    }
```

</details>
