## <img src="../assets/docuBadge (11).png" alt="" data-size="line"> hmac\_sha1 / hmac\_sha256 / hmac\_384 / hmac\_512 <a href="#xs-hmac_sha1-hmac_sha256-hmac_384-hmac_512" id="xs-hmac_sha1-hmac_sha256-hmac_384-hmac_512"></a>

`value|hmac_sha1:secret:hex_output`

<details>

<summary>Examples</summary>

```javascript
"data"|hmac_sha1:"secret":true     // Returns hex HMAC-SHA1 signature
"data"|hmac_sha1:"secret":false    // Returns base64 HMAC-SHA1 signature
```

</details>
