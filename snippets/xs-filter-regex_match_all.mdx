## <img src="../assets/docuBadge (11).png" alt="" data-size="line"> regex\_match\_all <a href="#xsregex_match_all" id="xsregex_match_all"></a>

`value|regex_match_all:pattern`

<details>

<summary>Examples</summary>

```javascript
"test123test456"|regex_match_all:"\\d+"    // Returns ["123", "456"]
"abc def"|regex_match_all:"[a-z]"          // Returns ["a", "b", "c", "d", "e", "f"]
```

</details>
