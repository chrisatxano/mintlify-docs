<details>

<summary>Examples</summary>

```javascript
encrypted_data|decrypt:"aes-128-cbc":"secret_key":""    // Returns decrypted data
encrypted_data|decrypt:"aes-256-cbc":"key":"iv"         // Returns decrypted data with IV
```
