## <img src="../assets/docuBadge (10).png" alt="" data-size="line"> Generate Password <a href="#xs-securitygeneratepw" id="xs-securitygeneratepw"></a>

```javascript
    security.create_password {
      character_count = 12
      require_lowercase = true
      require_uppercase = true
      require_digit = true
      require_symbol = true
      symbol_whitelist = "$#%&"
    } as generatedPassword
```
