## <img src="../assets/docuBadge (11).png" alt="" data-size="line"> number\_format <a href="#xsnumber_format" id="xsnumber_format"></a>

`value|number_format:decimals:dec_point:thousands_sep`

<details>

<summary>Examples</summary>

```javascript
1234.567|number_format:2:".":","`    // Returns "1,234.57"
1234.567|number_format:0:".":","`    // Returns "1,235"
```

</details>
