# MCP Call Tool

Executes a tool available on a remote MCP server

<table><thead><tr><th width="176.06878662109375">Parameter</th><th>Purpose</th></tr></thead><tbody><tr><td>url</td><td>The URL to access the MCP server</td></tr><tr><td>bearer token</td><td>If required, an authentication token to access the server</td></tr><tr><td>tool name</td><td>The name of the tool to call</td></tr><tr><td>args</td><td>The data that the tool requires, if any. This should usually be a JSON object.</td></tr></tbody></table>
