## <img src="../assets/docuBadge (11).png" alt="" data-size="line"> split <a href="#xssplit" id="xssplit"></a>

`value|split:delimiter`

<details>

<summary>Examples</summary>

```javascript
"a,b,c"|split:","    // Returns ["a", "b", "c"]
"test"|split:""      // Returns ["t", "e", "s", "t"]
```

</details>
