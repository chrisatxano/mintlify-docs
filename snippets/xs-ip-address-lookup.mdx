# <img src="../assets/docuBadge (11).png" alt="" data-size="line"> IP Address Lookup <a href="#xs-iplookup" id="xs-iplookup"></a>

```javascript
util.ip_lookup {
  value = "1.1.1.1"
} as ip_location
```

| Parameter | Purpose                               | Example                                     |
| --------- | ------------------------------------- | ------------------------------------------- |
| value     | IP address to look up                 | `"1.1.1.1"`, `"192.168.1.1"`, `$request.ip` |
| as        | Alias to reference the lookup results | `ip_location`, `visitor_location`           |

<details>

<summary>Example</summary>

```javascript
util.ip_lookup {
  value = $request|get:"ip"
} as visitor_info
```

* Performs geolocation lookup for an IP address

</details>
