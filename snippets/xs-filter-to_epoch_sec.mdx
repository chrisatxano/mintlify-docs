## <img src="../assets/docuBadge (11).png" alt="" data-size="line"> to\_seconds <a href="#xs-to_epoch_sec" id="xs-to_epoch_sec"></a>

`value|to_seconds:timezone`

<details>

<summary>Examples</summary>

```javascript
"2024-01-01"|to_seconds:"UTC"    // Returns seconds since Unix epoch
"now"|to_seconds:"UTC"           // Returns current seconds since Unix epoch
```

</details>
