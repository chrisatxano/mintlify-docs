## <img src="../assets/docuBadge (11).png" alt="" data-size="line"> url\_encode <a href="#xs-url_encode" id="xs-url_encode"></a>

`value|url_encode`

<details>

<summary>Examples</summary>

```javascript
"Hello World"|url_encode    // Returns "Hello%20World"
"test+file"|url_encode     // Returns "test%2Bfile"
```

</details>
