## <img src="../assets/docuBadge (4).png" alt="" data-size="line"> Add Record <a href="#xs-secretkey" id="xs-secretkey"></a>

```javascript
db.add user {
    data = {
        created_at: "now",
        name: $input.name,
        email: $input.email,
        password: $input.password
    }
} as $recordAdd
```

| Parameter | Purpose                                                    | Example                                     |
| --------- | ---------------------------------------------------------- | ------------------------------------------- |
| data      | Object containing the fields and values for the new record | `{name: "John", email: "john@example.com"}` |

<details>

<summary>Example</summary>

```javascript
// Create a new user record
db.add user {
    data = {
        created_at: "now",
        name: "John Doe",

    }
```
