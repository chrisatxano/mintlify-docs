## <img src="../assets/docuBadge (11).png" alt="" data-size="line"> append <a href="#xsappend" id="xsappend"></a>

`value|append:item:key`

<details>

<summary>Examples</summary>

```javascript
[1,2]|append:3                // Returns [1,2,3]
{"a":1}|append:2:"b"         // Returns {"a":1,"b":2}
```

</details>
