`value|secureid_encode:algorithm`

<details>

<summary>Examples</summary>

```javascript
"123"|secureid_encode:""           // Returns encrypted ID with default algorithm
"123"|secureid_encode:"aes-256"    // Returns encrypted ID with specified algorithm
```
