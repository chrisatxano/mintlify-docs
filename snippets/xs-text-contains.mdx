## <img src="../assets/docuBadge (10).png" alt="" data-size="line"> Contains <a href="#xs-textcontains" id="xs-textcontains"></a>

```javascript
    text.contains $myText {
      value = "Search text"
    } as $textFound
```

{% hint style="success" %}
This function is also available as a **case-insensitive version** as:

```
text.icontains
```
{% endhint %}
