# <img src="../assets/docuBadge (11).png" alt="" data-size="line"> Realtime Event <a href="#xs-realtime-event" id="xs-realtime-event"></a>

```javascript
api.realtime_event {
  channel = ""
  data = ""
  auth_table = ""
  auth_id = ""
}
```

| Parameter   | Purpose              | Example                               |
| ----------- | -------------------- | ------------------------------------- |
| channel     | Event channel name   | `"notifications"`, `"user_123"`       |
| data        | Event payload        | `{type: "message", content: "Hello"}` |
| auth\_table | Authorization table  | `"users"`, `"organizations"`          |
| auth\_id    | Authorized entity ID | `"123"`, `$user.id`                   |

<details>

<summary>Example</summary>

```javascript
api.realtime_event {
  channel = "user:"|add:$user.id
  data = {
    type: "notification",
    message: "New message received",
    timestamp: $now
  }
  auth_table = "users"
  auth_id = $user.id
}
```

* Sends realtime events to clients

</details>
