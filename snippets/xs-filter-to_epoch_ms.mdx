## <img src="../assets/docuBadge (11).png" alt="" data-size="line"> to\_timestamp <a href="#xs-to_epoch_ms" id="xs-to_epoch_ms"></a>

`value|to_timestamp:timezone`

<details>

<summary>Examples</summary>

```javascript
"2024-01-01"|to_timestamp:"UTC"    // Returns milliseconds since Unix epoch
"now"|to_timestamp:"UTC"           // Returns current milliseconds since Unix epoch
```

</details>
