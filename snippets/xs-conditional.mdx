## <img src="../assets/docuBadge (10).png" alt="" data-size="line"> Conditional If/Then/Else Statement <a href="#xs-conditional" id="xs-conditional"></a>

```javascript
    conditional {
      if (`conditions`) {
        // Functions go here
      }
      else {
        // Functions go here
        }
    }
```

<table><thead><tr><th width="175">Parameter</th><th>Purpose</th><th>Example</th></tr></thead><tbody><tr><td>conditions</td><td>This is where the condition(s) you want to check will live.</td><td><pre><code>`"a" == 1 &#x26;&#x26; "b" == 2 || "c" == 3`
</code></pre></td></tr></tbody></table>

<details>

<summary>Example</summary>

```javascript
    conditional {
      if (`"a" == 1 && "b" == 2 || "c" == 3`) {
        util.sleep {
          value = 3
        }
      }
    
      else {
        util.sleep {
          value = 30
        }
      }
    }
```
