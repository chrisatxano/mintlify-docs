## <img src="../assets/docuBadge (11).png" alt="" data-size="line"> unset <a href="#xsunset" id="xsunset"></a>

`value|unset:key`

<details>

<summary>Examples</summary>

```javascript
{"name":"John","age":30}|unset:"age"   // Returns {"name":"John"}
{"name":"John"}|unset:"address"        // Returns {"name":"John"}
```

</details>
